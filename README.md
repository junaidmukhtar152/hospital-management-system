# 📌 Overview

The **Hospital Management System** is a full-stack web application designed to simplify and automate hospital management tasks such as:

- 👨‍⚕️ Doctor Management
- 🧑‍🤝‍🧑 Patient Management
- 📅 Appointment Booking
- 💊 Medicine & Prescription Records
- 🛏️ Room & Ward Management
- 📋 Patient Reports
- 🔐 Authentication & Security

This project helps hospitals improve efficiency, reduce paperwork, and manage data digitally.

---

# 🚀 Features

## 👨‍⚕️ Doctor Panel
- Add new doctors
- Manage doctor information
- View patient appointments
- Update schedules

## 🧑‍🤝‍🧑 Patient Panel
- Patient registration
- Book appointments
- View medical records
- Prescription history

## 🛡️ Admin Panel
- Manage doctors & patients
- Control appointments
- Monitor system activities
- Secure login system

## 📱 Responsive Design
- Mobile Friendly 📱
- Tablet Responsive 💻
- Smooth UI/UX ✨
